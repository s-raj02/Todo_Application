# ✅ Todo Application

## 📌 Project Overview

Todo Application is a full-stack task management system designed to help users organize, manage, and track their daily tasks efficiently. The application provides a clean and responsive interface where users can create, update, complete, and delete tasks in real time.

The project demonstrates full-stack web development concepts including CRUD operations, RESTful API development, database integration, state management, and responsive UI design using modern web technologies.

---

## 🎯 Objectives

* Build a full-stack task management application.
* Implement CRUD (Create, Read, Update, Delete) operations.
* Store and manage tasks efficiently using a database.
* Provide a responsive and user-friendly interface.
* Improve productivity through task organization.
* Demonstrate RESTful API integration between frontend and backend.

## ⚙️ Project Workflow

### 1. Task Creation

Users can create and save new tasks.

### 2. Task Retrieval

Stored tasks are fetched from the database and displayed on the dashboard.

### 3. Task Update

Users can edit existing tasks and update their information.

### 4. Task Completion

Tasks can be marked as completed or pending.

### 5. Task Deletion

Users can remove completed or unnecessary tasks.

### 6. Database Synchronization

All task operations are synchronized with MongoDB through REST APIs.

## ▶️ How to Run

### Clone Repository

```bash
git clone https://github.com/s-raj02/Todo_Application.git
```

### Navigate to Project Directory

```bash
cd Todo_Application
```

### Install Frontend Dependencies

```bash
cd client
npm install
```

### Install Backend Dependencies

```bash
cd server
npm install
```

### Configure Environment Variables

Create a `.env` file and add:

```env
MONGODB_URL=your_mongodb_connection_string
PORT=5000
```

### Run Backend

```bash
cd server
npm run dev
```

### Run Frontend

```bash
cd client
npm start
```
