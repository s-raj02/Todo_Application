# 📌 Todo Application

A simple and efficient Todo Application backend built using **Node.js, Express.js, and MongoDB**.  
This project demonstrates core **CRUD operations**, REST API design, and MongoDB integration using Mongoose.

---

## 🚀 Features

- Create new tasks  
- Read all tasks  
- Update existing tasks  
- Delete tasks  
- MongoDB database integration  
- RESTful API architecture  
- Lightweight backend system  

---

## 🛠️ Tech Stack

- Node.js  
- Express.js  
- MongoDB  
- Mongoose  
- JavaScript (ES6)

## 📡 API Endpoints
Method	Endpoint	Description

GET	/todos	Get all tasks

POST	/todos	Create task

PUT	/todos/:id	Update task

DELETE	/todos/:id	Delete task
